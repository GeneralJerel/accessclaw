## Key Technical Decisions

| Decision | Rationale |
|----------|-----------|
| TanStack Router (file-based) | Clean route organization, type-safe, auto code-splitting |
| Tailwind CSS v4 | Utility-first with custom Liquid Glass theme tokens; `@tailwindcss/vite` plugin |
| CopilotKit for agent UI | Purpose-built for AI copilot interfaces; handles AG-UI consumption (deferred to Phase 2) |
| Custom Chat Sidebar | Liquid Glass-styled sidebar ready for CopilotKit integration when runtime is available |
| Data-first (mock) | Frontend-first development with `/data` directory; backend deferred until UX is validated |
| Atomic design | Composable, testable components; scales well for a dashboard-heavy app |
